📋 Recurring Contracts — System Design

Persistent buyer relationships with 4 tiers. Consistent delivery unlocks better bonuses, bigger orders, and disaster grace periods.

1. New Data Types

Buyer new type

type Buyer = { id: string name: string emoji: string cropIds: string[] // what they buy tier: BuyerTier // 'new'|'regular'|'preferred'|'exclusive' deliveryStreak: number // consecutive on-time totalDeliveries: number missedInARow: number // 2 = contract cancelled unlockedDay: number // when player can access }

RecurringContract new type

type RecurringContract = { id: string buyerId: string cropId: string amountPerDelivery: number // kg frequencyDays: 7|14|30 priceBonus: number // e.g. 0.27 = +27% nextDeliveryDay: number deliveryWindowDays: number // days to fulfill durationSeasons: number // 1–4 endDay: number active: boolean graceDaysRemaining: number }

4. Disaster Grace Period

When a weather insurance claim triggers (frost kill, drought wipeout, hail damage), active recurring contracts automatically receive a grace period equal to the buyer's tier grace window. During grace: the delivery window is extended, no streak penalty applies, and missedInARow is not incremented.

TierGrace periodRequires
NewNo grace—
Regular+3 daysActive weather insurance
Preferred+7 daysActive weather insurance
Exclusive+14 daysAny active insurance

This makes insurance genuinely valuable for contract farmers — not just yield protection but relationship protection.

5. The 8 Buyers

🥖 Local Bakery
Wheat, Corn • Weekly • Small orders
Available from day 1. Perfect starter — forgiving window, low amounts. Great for building your first streak.
🍽️ City Restaurant
Premium crops (lavender, saffron, vanilla) • Biweekly
Unlocks at reputation 40. High price bonus. Requires premium-tier crops — can't substitute cheap alternatives.
🏭 Export Processor
Bulk grains • Monthly • Large volumes
Unlocks at reputation 60. Monthly cadence gives planning time. Large orders demand serious farm scale.
🥛 Dairy Distributor
Milk, Cheese, Butter • Weekly
Unlocks after first dairy building. Tightly tied to animal production output. Milk grade affects accepted quantity.
🌿 Organic Market
Any certified-organic crop • Weekly
Unlocks when first building reaches Organic certification. Highest price bonus but strict: loses contract if certification drops.
🛒 Regional Supermarket
Mixed crops (rotates seasonally) • Biweekly
Unlocks day 30. Accepts a variety of crops each season — good for diversified farms. Lowest penalty rate.
🍺 Distillery
Barley, Corn, Sugarbeet • Monthly
Unlocks at reputation 50. Accepts lower-grade crops at decent prices. Good outlet for excess harvest.
🐾 Pet Food Co.
Any crop (low-grade ok) • Weekly
Available from day 1. Lowest bonus (+15%) but accepts anything including spoiled/damaged stock. Safety net buyer.

7. Where It Lives in the UI

New "Supply" sub-tab added to the Economy screen alongside Market / Auto-Sell / Stats / Futures / Orders.

Buyers list — shows all 8 buyers, tier badge, current streak, and whether a contract is active.

Active contract card — per buyer, shows streak pips, next delivery countdown, stock check (do you have enough?), grace period status.

Delivery calendar in Calendar screen — recurring deliveries appear as deadline markers alongside seasonal events.

HUD warning strip — existing warning strip system gets a new "⚠️ Delivery due in 2d – Dairy Distributor" entry when within 3 days of a window closing.
