🚛 Truck System (Vehicle + Trailer)

Trucks are purchased separately from trailers. This allows future functionality for pickups/semis beyond transport.

Standalone Vehicles
🚙 Pickup — $12,000 · 500 units
🚛 Dump Truck — $18,000 · for future
Vehicle + Trailer Combos
🚙 Pickup + Trailer — Pickup ($12k) + Trailer ($5–12k)
🚛 Semi + Trailer — Semi ($25k) + Trailer ($8–18k)
Trailer Types
Small Trailer — $5,000 · 1,500 units · fits Pickup
Standard Trailer — $8,000 · 3,500 units · fits Pickup or Semi
Large Trailer — $15,000 · 8,000 units · Semi only

🔑 Key Rules

• Till required before plant — parcel must be tilled: true to accept a crop
• Harvest requires combine — no combine = hire contractor or crop sits in field
• One job per tractor — assigning a new job when busy is blocked
• Attachment must match tractor size — small attachment fits small+medium tractor; large only fits large
• Trailers are separate purchases — pickup/semi alone can do other things; need trailer to transport
• Contractor = instant — hires complete the operation same day, no machine required
• Transport optional — without a truck, crops sell at -12% (contractor cut built into sell price)
